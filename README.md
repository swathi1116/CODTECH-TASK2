DESCRIPTION :

I successfully completed a Speech to Text Converter using Automatic Speech Recognition (ASR) techniques as part of my AI Internship at CODTECH IT SOLUTIONS. The main objective of this task was to develop a Python application capable of converting spoken audio into readable text automatically. This project helped me understand the fundamentals of speech recognition, audio processing, and the practical implementation of Artificial Intelligence in Natural Language Processing.

For this project, I used Python as the programming language and Visual Studio Code (VS Code) as the development environment. VS Code provided an efficient platform for writing, executing, and debugging the Python code. I also used Python's package manager (pip) to install the required speech recognition libraries and dependencies.

The primary libraries used in this project were SpeechRecognition and PyAudio. The SpeechRecognition library was used to process audio input and convert spoken words into text using Google's Speech Recognition engine. PyAudio was used to access the microphone and capture voice input from the user. These libraries made it possible to build a simple yet effective speech recognition application.

The task began by importing the required libraries and initializing the speech recognizer. The program then captured audio input either through the system microphone or from an audio file. The captured speech was processed using Google's Speech Recognition API, which converted the spoken words into text. The recognized text was then displayed in the terminal for the user.

During the implementation, I also handled possible errors such as unclear speech, background noise, and network-related issues by using Python's exception handling techniques. This made the application more reliable and user-friendly.

Through this project, I gained practical experience in speech recognition, audio processing, Python programming, and AI-based language processing techniques. I also learned how modern virtual assistants and voice-controlled systems convert human speech into text. This task strengthened my understanding of Artificial Intelligence applications and improved my programming skills in developing real-world AI solutions.
